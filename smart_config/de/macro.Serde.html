Macro Serde

Source 
macro_rules! Serde {
    (*) => { ... };
    ($($expecting:tt),+ $(,)?) => { ... };
}
Expand description

Constructor of Serde types / instances.

The macro accepts a comma-separated list of expected basic types from the following set: bool, int, float, str, array, object. As a shortcut, Serde![*] signals to accept any input.

    #[config(with = Serde![int, str, object])]
